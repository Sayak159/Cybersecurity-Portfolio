# Hash File Checker

## Descrição

Ferramenta desenvolvida em Python para calcular os hashes MD5 e SHA-256 de arquivos.

## Funcionalidades

- Cálculo de MD5
- Cálculo de SHA-256
- Leitura de qualquer tipo de arquivo
- Tratamento de erros

## Tecnologias

- Python 3
- hashlib
